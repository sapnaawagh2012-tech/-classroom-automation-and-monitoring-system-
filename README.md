 Classroom Automation and Monitoring System

IoT-based smart classroom automation project using sensors and ESP32.

 Overview
This project automates classroom monitoring and control using ESP32 and IoT sensors.

 Features
- Automatic light control
- Temperature monitoring
- Smart energy saving
- Sensor-based automation

 Components Used
- ESP32
- DHT11 Sensor
- PIR Sensor
- Relay Module

 Technologies
- Arduino IDE
- Embedded C
- IoT

Future Scope
- Mobile app integration
- Cloud monitoring
- Voice control
